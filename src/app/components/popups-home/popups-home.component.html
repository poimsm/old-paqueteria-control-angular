Entrego el pedido
Situación: El rider entrego el pedido pero no informó al sistema debido a una falla (su teléfono quedo sin baterias, se le terminó el plan de datos, etc.).
Acción: El pedido se marcará como entregado y el rider será reiniciado para recibir nuevos pedidos.
No completo el pedido
Situación: El rider por algun motivo (accidente, robo, etc.) no logró completar el pedido.
Acción: El pedido quedará anulado y el rider será reiniciado para recibir nuevos pedidos.
El cliente no confirmó el pedido
Situación: El cliente ha demorado mucho (5 min o más) en confirmar el pedido.
Acción: El rider será reiniciado para recibir nuevos pedidos.
